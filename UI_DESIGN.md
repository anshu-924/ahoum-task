## Responsive Design

### Desktop (> 768px)
- Grid layout: 3 columns for session cards
- Side-by-side forms
- Full navigation bar

### Tablet (768px - 480px)
- Grid layout: 2 columns
- Stacked forms
- Compact navigation

### Mobile (< 480px)
- Grid layout: 1 column
- Vertical forms
- Hamburger menu (can be added)

## Interactions

### Hover States
- Buttons: Darker shade
- Cards: Lift up (transform: translateY(-4px))
- Links: Color change to grey

### Loading States
- Spinner or "Loading..." text
- Disabled buttons during actions
- Skeleton screens (can be added)

### Error States
- Red text below form fields
- Error messages in red boxes
- Form validation on submit

### Success States
- Green checkmark
- Success messages in green boxes
- Auto-redirect after actions
